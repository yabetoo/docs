Vous pouvez commencer à utiliser votre compte Yabetoo dans des environnements de test dès sa création. Dans un environnement sandbox, vous pouvez simuler l'utilisation de toutes les fonctionnalités de Yabetoo sans déplacer d'argent réel. Une fois votre compte activé, vous pouvez accepter des paiements, créer des comptes supplémentaires et gérer une équipe.

## Configuration

La première étape vers une documentation de classe mondiale est la configuration de vos environnements d'édition.
