L'API Yabetoo supporte actuellement le Franc CFA (XAF) pour traiter les paiements en Afrique Centrale.

## Devise supportée

| Devise | Code | Région | Description |
|--------|------|--------|-------------|
| Franc CFA | `XAF` | Afrique Centrale | Devise pour les pays de la CEMAC |

## Spécifier la devise

Incluez le paramètre `currency` lors de la création d'une intention de paiement :

```bash
curl -X POST https://pay.sandbox.yabetoopay.com/v1/payment-intents \
  -H "Authorization: Bearer sk_test_XXXXXXXXXXXXXXXXXXXXXXXX" \
  -H "Content-Type: application/json" \
  -d '{
    "amount": 5000,
    "currency": "xaf"
  }'
```

## Format des montants

Les montants sont spécifiés en Francs (la plus petite unité) :

| Devise | Unité | Exemple |
|--------|-------|---------|
| `XAF` | Franc | 5000 = 5 000 XAF |
