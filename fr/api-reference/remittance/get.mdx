---
title: "Récupérer un transfert"
description: "Récupérez un transfert par son ID pour vérifier son statut et ses détails."
---

## Endpoint

```bash
GET https://pay.sandbox.yabetoopay.com/v1/remittance/{id}   # Sandbox
GET https://pay.api.yabetoopay.com/v1/remittance/{id}       # Production
```

## Authentification

Utilisez votre clé secrète dans l'en-tête `Authorization` :

```bash
Authorization: Bearer YOUR_SECRET_KEY
```

## Paramètres de chemin

| Paramètre | Type | Description |
|-----------|------|-------------|
| `id` | `string` | L'ID unique du transfert |

### Statuts du transfert

| Statut | Description |
|--------|-------------|
| `pending` | Le transfert est en attente de traitement |
| `processing` | Le transfert est en cours de traitement |
| `succeeded` | Le transfert a réussi |
| `failed` | Le transfert a échoué |

### 404 Not Found

```json
{
  "message": "Remittance not found"
}
```

### 401 Non autorisé

```json
{
  "message": "Unauthorized"
}
```
