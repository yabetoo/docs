## Prérequis

Avant de commencer, assurez-vous d'avoir :

<CardGroup cols={2}>
  <Card title="Compte Yabetoo" icon="user">
    [Créez un compte](https://app.yabetoo.com) si ce n'est pas déjà fait
  </Card>
  <Card title="Clés API" icon="key">
    Obtenez vos clés API de test depuis le Dashboard
  </Card>

<Warning>
  Ne partagez jamais votre clé API secrète publiquement. Utilisez des variables d'environnement pour les stocker en toute sécurité.
</Warning>

## Étape 1 : Obtenez vos clés API

<Steps>
  <Step title="Connectez-vous à votre Dashboard">
    Allez sur [app.yabetoo.com](https://app.yabetoo.com) et connectez-vous à votre compte
  </Step>
  <Step title="Accédez aux clés API">
    Cliquez sur **Paramètres** > **Clés API** dans la barre latérale
  </Step>
  <Step title="Copiez vos clés de test">
    Copiez votre clé secrète de test (commence par `sk_test_`)
  </Step>
</Steps>

<Info>
  Les clés de test vous permettent de simuler des paiements sans argent réel. Utilisez-les pendant le développement.
</Info>

## Étape 3 : Créez votre premier paiement

### Option A : Page de paiement hébergée (Recommandé)

Le moyen le plus rapide d'accepter des paiements. Redirigez vos clients vers une page hébergée par Yabetoo.

<CodeGroup>

```bash curl
curl -X POST https://buy.api.yabetoopay.com/v1/sessions \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer sk_test_XXXXXXXXXXXXXXXXXXXXXXXX" \
  -d '{
    "total": 5000,
    "currency": "xaf",
    "accountId": "acct_xxxxxxxx",
    "successUrl": "https://votre-site.com/succes",
    "cancelUrl": "https://votre-site.com/annuler",
    "items": [{
      "productId": "prod_123",
      "productName": "Plan Premium",
      "quantity": 1,
      "price": 5000
    }]
  }'
```

```javascript Node.js
import { Yabetoo } from '@yabetoopay/node';

const yabetoo = new Yabetoo('sk_test_XXXXXXXXXXXXXXXXXXXXXXXX');

const session = await yabetoo.sessions.create({
  accountId: 'acct_xxxxxxxx',
  total: 5000,
  currency: 'xaf',
  successUrl: 'https://votre-site.com/succes',
  cancelUrl: 'https://votre-site.com/annuler',
  items: [
    {
      productId: 'prod_123',
      productName: 'Plan Premium',
      quantity: 1,
      price: 5000,
    },
  ],
});

// Redirigez le client vers session.url
console.log(session.url);
```

```python Python
from yabetoo import Yabetoo
from yabetoo.models.checkout import CreateCheckoutSession, CheckoutItem

yabetoo = Yabetoo('sk_test_XXXXXXXXXXXXXXXXXXXXXXXX')

session = yabetoo.sessions.create(CreateCheckoutSession(
    account_id='acct_xxxxxxxx',
    total=5000,
    currency='xaf',
    success_url='https://votre-site.com/succes',
    cancel_url='https://votre-site.com/annuler',
    items=[
        CheckoutItem(
            product_id='prod_123',
            product_name='Plan Premium',
            quantity=1,
            price=5000
        )
    ]
))

# Redirigez le client vers session.url
print(session.url)
```

```php PHP
<?php
require_once('vendor/autoload.php');

$yabetoo = new \Yabetoo\YabetooClient('sk_test_XXXXXXXXXXXXXXXXXXXXXXXX');

$session = $yabetoo->sessions->create([
  'accountId' => 'acct_xxxxxxxx',
  'total' => 5000,
  'currency' => 'xaf',
  'successUrl' => 'https://votre-site.com/succes',
  'cancelUrl' => 'https://votre-site.com/annuler',
  'items' => [
    [
      'productId' => 'prod_123',
      'productName' => 'Plan Premium',
      'quantity' => 1,
      'price' => 5000
    ]
  ]
]);

// Redirigez le client vers $session->url
echo $session->url;
?>
```

</CodeGroup>

### Option B : Intégration personnalisée

Pour un contrôle total sur l'expérience de paiement, utilisez l'API Payment Intent.

<CodeGroup>

```bash curl
# Étape 1 : Créer l'intention de paiement
curl https://pay.sandbox.yabetoopay.com/v1/payment-intents \
  -X POST \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer sk_test_XXXXXXXXXXXXXXXXXXXXXXXX" \
  -d '{
    "amount": 5000,
    "currency": "xaf",
    "description": "Abonnement Plan Premium"
  }'

# Étape 2 : Confirmer avec le moyen de paiement du client
curl https://pay.sandbox.yabetoopay.com/v1/payment-intents \
  -X POST \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer sk_test_XXXXXXXXXXXXXXXXXXXXXXXX" \
  -d '{
    "client_secret": "pi_xxx_secret_yyy",
    "first_name": "Jean",
    "last_name": "Dupont",
    "receipt_email": "jean@example.com",
    "payment_method_data": {
      "type": "momo",
      "momo": {
        "country": "cg",
        "msisdn": "242123456789",
        "operator_name": "mtn"
      }
    }
  }'
```

```javascript Node.js
import { Yabetoo } from '@yabetoopay/node';

const yabetoo = new Yabetoo('sk_test_XXXXXXXXXXXXXXXXXXXXXXXX');

// Étape 1 : Créer l'intention de paiement
const intent = await yabetoo.paymentIntents.create({
  amount: 5000,
  currency: 'xaf',
  description: 'Abonnement Plan Premium',
});

// Étape 2 : Confirmer avec le moyen de paiement du client
const payment = await yabetoo.paymentIntents.confirm({
  client_secret: intent.client_secret,
  first_name: 'Jean',
  last_name: 'Dupont',
  receipt_email: 'jean@example.com',
  payment_method_data: {
    type: 'momo',
    momo: {
      country: 'cg',
      msisdn: '242123456789',
      operator_name: 'mtn',
    },
  },
});

console.log(payment.status); // "succeeded"
```

```python Python
from yabetoo import Yabetoo
from yabetoo.models.payment import (
    CreateIntentRequest,
    ConfirmIntentRequest,
    PaymentMethodData,
    MomoData
)

yabetoo = Yabetoo('sk_test_XXXXXXXXXXXXXXXXXXXXXXXX')

# Étape 1 : Créer l'intention de paiement
intent = yabetoo.payment_intents.create(CreateIntentRequest(
    amount=5000,
    currency='xaf',
    description='Abonnement Plan Premium'
))

# Étape 2 : Confirmer avec le moyen de paiement du client
payment = yabetoo.payment_intents.confirm(
    payment_id=intent.id,
    data=ConfirmIntentRequest(
        client_secret=intent.client_secret,
        first_name='Jean',
        last_name='Dupont',
        receipt_email='jean@example.com',
        payment_method_data=PaymentMethodData(
            type='momo',
            momo=MomoData(
                country='cg',
                msisdn='242123456789',
                operator_name='mtn'
            )
        )
    )
)

print(payment.status)  # "succeeded"
```

```php PHP
<?php
require_once('vendor/autoload.php');

$yabetoo = new \Yabetoo\YabetooClient('sk_test_XXXXXXXXXXXXXXXXXXXXXXXX');

// Étape 1 : Créer l'intention de paiement
$intent = $yabetoo->paymentIntents->create([
  'amount' => 5000,
  'currency' => 'xaf',
  'description' => 'Abonnement Plan Premium'
]);

// Étape 2 : Confirmer avec le moyen de paiement du client
$payment = $yabetoo->paymentIntents->confirm([
  'client_secret' => $intent->client_secret,
  'first_name' => 'Jean',
  'last_name' => 'Dupont',
  'receipt_email' => 'jean@example.com',
  'payment_method_data' => [
    'type' => 'momo',
    'momo' => [
      'country' => 'cg',
      'msisdn' => '242123456789',
      'operator_name' => 'mtn'
    ]
  ]
]);

echo $payment->status; // "succeeded"
?>
```

## Étape 5 : Passez en production

Quand vous êtes prêt à accepter de vrais paiements :

<Steps>
  <Step title="Complétez la vérification du compte">
    Soumettez vos documents d'entreprise dans le Dashboard
  </Step>
  <Step title="Passez aux clés de production">
    Remplacez `sk_test_` par `sk_live_` dans votre code
  </Step>
  <Step title="Mettez à jour les endpoints webhook">
    Pointez les webhooks vers votre serveur de production
  </Step>
</Steps>

<Check>
  Félicitations ! Vous êtes maintenant prêt à accepter des paiements avec Yabetoo.
</Check>
