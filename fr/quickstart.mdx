## Configuration de votre environnement

Apprenez à mettre à jour votre documentation localement et à la déployer publiquement.

### Édition et prévisualisation

<AccordionGroup>
  <Accordion icon="github" title="Cloner votre documentation en local">
    Pendant le processus d'intégration, nous avons créé un dépôt sur votre Github avec
    le contenu de votre documentation. Vous pouvez trouver ce dépôt sur notre
    [tableau de bord](https://dashboard.mintlify.com). Pour cloner le dépôt
    localement, suivez ces
    [instructions](https://docs.github.com/fr/repositories/creating-and-managing-repositories/cloning-a-repository)
    dans votre terminal.
  </Accordion>
  <Accordion icon="rectangle-terminal" title="Prévisualiser les changements">
    La prévisualisation vous aide à vous assurer que vos modifications apparaissent comme prévu. Nous avons créé une
    interface en ligne de commande pour afficher ces changements localement.
    1. Installez le
    [CLI Mintlify](https://www.npmjs.com/package/mintlify) pour prévisualiser les
    changements de documentation localement avec cette commande : ``` npm i -g mintlify ```
    2. Exécutez la commande suivante à la racine de votre documentation (où se trouve
    `docs.json`) : ``` mintlify dev ```
    <Note>
      Si vous utilisez actuellement le fichier de configuration legacy ```mint.json```, veuillez mettre à jour le CLI Mintlify :

      ```npm i -g mintlify@latest```
      Et exécutez la nouvelle commande de mise à niveau dans votre dépôt de documentation :

      ```mintlify upgrade```
      Vous devriez maintenant utiliser le nouveau fichier de configuration ```docs.json```. N'hésitez pas à supprimer le fichier ```mint.json``` de votre dépôt.
    </Note>
  </Accordion>
</AccordionGroup>

### Déployer vos changements

<AccordionGroup>

<Accordion icon="message-bot" title="Installer notre application Github">
  Notre application Github déploie automatiquement vos modifications sur votre site de documentation,
  vous n'avez donc pas besoin de gérer les déploiements vous-même. Vous pouvez trouver le lien d'installation sur
  votre [tableau de bord](https://dashboard.mintlify.com). Une fois que le bot a été
  installé avec succès, il devrait y avoir une coche à côté du hash du commit
  du dépôt.
</Accordion>
<Accordion icon="rocket" title="Pousser vos changements">
  [Committez et poussez vos changements sur
  Git](https://docs.github.com/fr/get-started/using-git/pushing-commits-to-a-remote-repository#about-git-push)
  pour que vos modifications soient mises à jour sur votre site de documentation. Si vous poussez et ne voyez pas que
  l'application Github a déployé vos changements avec succès, vous pouvez également
  mettre à jour manuellement votre documentation via notre [tableau de bord](https://dashboard.mintlify.com).
</Accordion>

</AccordionGroup>
