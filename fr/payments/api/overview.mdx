title: "Vue d'ensemble"
description: "Utilisez l'API Yabetoo pour accepter des paiements dans votre application"
---

Yabetoo fonctionne sur un modèle de paiement en deux étapes. Cela signifie que, pour traiter un paiement, vous devez d'abord créer une intention de paiement, puis confirmer cette intention pour capturer le montant.

Cette approche offre une plus grande flexibilité, vous permettant par exemple de vérifier les fonds disponibles ou d'obtenir l'autorisation d'un utilisateur avant de finaliser une transaction. Voici comment cela fonctionne, étape par étape :
