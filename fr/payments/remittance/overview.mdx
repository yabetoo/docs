La fonctionnalité de transfert d'argent permet aux entreprises partenaires d'agir en tant qu'opérateurs de transfert d'argent en envoyant des fonds depuis leur compte professionnel Yabetoo vers des bénéficiaires finaux (clients, particuliers ou entreprises) pour le compte de leurs propres utilisateurs.

Contrairement à la fonctionnalité de transfert standard — qui permet aux marchands d'envoyer des fonds depuis leur compte pour leur propre usage — le transfert d'argent est conçu pour les entreprises qui offrent des services de transfert d'argent à des tiers.

## Objectif et avantages

L'API de transfert d'argent vous permet de :

- Transférer des fonds **pour le compte d'un utilisateur tiers** (agent, client ou bénéficiaire final).
- Enrichir les transferts avec des **métadonnées personnalisées** (référence interne, ID utilisateur, canal de transfert, etc.).
- **Suivre l'état des transactions** en temps réel (`en attente`, `en cours`, `réussi`, `échoué`).
- Recevoir des **notifications webhook** lors des changements d'état des transferts.
- Gérer les opérations de transfert via une **API sécurisée et flexible**.

## Cas d'utilisation courants

| Cas d'utilisation                   | Description                                                                                                                |
| ----------------------------------- | -------------------------------------------------------------------------------------------------------------------------- |
| **Plateforme de transfert d'argent** | Une application fintech permet aux utilisateurs d'envoyer de l'argent à leurs proches. La plateforme utilise l'API de transfert pour traiter les transactions. |
| **Paiements de masse**              | Une entreprise ou une ONG distribue des fonds à plusieurs bénéficiaires (par ex., salaires, commissions, aide).            |
| **Transfert par agent en personne**  | Un agent physique utilise une interface connectée à Yabetoo pour initier des transferts pour les clients sur place.        |
| **Paiement de service pour compte de** | Le partenaire paie des factures ou des services pour ses utilisateurs finaux (par ex., frais de scolarité, électricité).  |

## À qui s'adresse cette fonctionnalité ?

La fonctionnalité de transfert d'argent est disponible pour les **partenaires approuvés** dont le modèle d'affaires inclut le traitement des paiements pour le compte de tiers, tels que :

- Les entreprises fintech ou les fournisseurs de services de transfert d'argent
- Les plateformes de paiement et de services financiers
- Les coopératives ou institutions de microfinance
- Les ONG et programmes de développement
