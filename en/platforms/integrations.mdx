## Why Use Platform Integrations?

### Quick Setup
Install and configure in minutes without writing any code. Our plugins handle all the complexity of payment processing.

### Automatic Updates
Receive automatic updates with new features, security patches, and improvements without manual intervention.

### Native Experience
Our integrations are built to feel native to each platform, following their UI patterns and best practices.

### Mobile Money Ready
All integrations support Mobile Money payments (MTN, Airtel) out of the box, perfect for African markets.

## Getting Started

1. **Choose your platform** - Select the integration guide for your e-commerce platform
2. **Install the plugin** - Follow our step-by-step installation guide
3. **Configure API keys** - Add your Yabetoo API keys from the dashboard
4. **Test in sandbox** - Verify everything works with test transactions
5. **Go live** - Switch to production and start accepting payments
