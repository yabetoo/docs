## Intégrez les paiements Yabetoo facilement avec nos SDK
Yabetoo simplifie l’intégration des paiements pour les développeurs avec une gamme complète de SDKs conçus pour différents langages. Que vous soyez développeur PHP, JavaScript, Python, ou autre, nous avons un SDK adapté à vos besoins pour que vous puissiez vous concentrer sur ce qui compte le plus : développer vos applications.

## SDKs côté serveur

Les SDK côté serveur de Yabetoo réduisent la charge de travail nécessaire pour utiliser les API REST de Yabetoo. Retrouvez les instructions d’installation et des exemples de requêtes API dans notre guide d’introduction aux SDK côté serveur.

## Commencez dès aujourd’hui !

Que vous construisiez une boutique en ligne, une plateforme de services ou une application SaaS, nos SDKs vous permettent d’accepter des paiements en toute simplicité. Consultez la documentation, installez le SDK de votre choix, et commencez à intégrer Yabetoo dès aujourd’hui.
